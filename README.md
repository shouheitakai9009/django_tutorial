## なぜやるのか
サーバーサイドはJava seasar2 / CakePHP3 / Expressを扱ってきたが、最近はめっきりReact一本。
いつしかフロントエンドエンジニアと呼ばれ、Railsは全くわからずフロント専門みたいになってしまったのが悔しい。

- Python3力を鍛える
